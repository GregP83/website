Hypermark

My most advanced and complicated project I did mostly on my own:
  • User Interface (*with DockPanel Suite)
  • DrawPanel - element on which elements can be drawn and manipulated
  • Manipulation of DrawObjects (moving, resizing, rotating)
  • Plug-in architecture for additional DrawObjects
  • Basic DrawObjects: Line, Rectangle, Circle, Polygon
  • Optional DrawObjects: Text, QR code, PLT file import
  • Custom fonts for Text (including bold and italic). Left, right, and stretch alignment. Option to manually set the distance between lines and characters. Etc.
  • Possibility to group few objects into new object and ungroup after modification
  • Convert any of the objects into polygon
  • Undo-Redo functionality
  • Lens distortion correction for all vectors on DrawPanel
  • Laser path simulation

Different components of Hypermark were later used in my other projects.



*Hypermark used DockPanel Suite by Weifen Luo - it is a WinForm library for emulating the docking system of Visual Studio.
