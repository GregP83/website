ICS-6000 Tablet App

ICS-6000 Tablet App allows to directly and securely control Thermo Scientific™ Dionex™ ICS-6000™ HPIC™ system, view the current status of parameters, display a real-time plot of signal data, and offer access to the ICS-6000 user manual, Troubleshooting Knowledgebase and Consumable Installation Guides.

Tablet app is a Universal Windows Platform application written for Microsoft Store. It communicates with the Chromeleon Mobile Devices service to allow control over the ICS-6000 instruments.

In this project I was a Lead Developer, that means I was responsible for everything.
